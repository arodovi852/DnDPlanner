Criterio 1c) Identificación de necesidades
Nivel 	Descripción 	Puntuación
Excelente 	Se identifica claramente una necesidad real con evidencias sólidas (entrevistas, datos, análisis). Se definen usuarios objetivo con detalle. El problema está bien delimitado y justificado. 	10
Bien 	Se identifica una necesidad pero las evidencias son limitadas. Los usuarios objetivo están definidos de forma genérica. El problema está parcialmente justificado. 	6
Insuficiente 	No se identifica claramente una necesidad o carece de justificación. No se definen usuarios objetivo o la definición es irrelevante. 	0
Criterio 1d) Oportunidades de negocio
Nivel 	Descripción 	Puntuación
Excelente 	Se realiza un análisis de mercado completo con competidores identificados. Se articula claramente la propuesta de valor diferencial. Se valora el potencial con argumentos sólidos. 	10
Bien 	Se mencionan competidores pero sin análisis profundo. La propuesta de valor es genérica. El potencial se menciona pero sin fundamentar. 	6
Insuficiente 	No se analiza el mercado ni la competencia. No se diferencia de soluciones existentes. No se valora el potencial del proyecto. 	0
Criterio 1e) Tipo de proyecto
Nivel 	Descripción 	Puntuación

Insuficiente 	No se define el tipo de proyecto o la definición es incorrecta. No hay arquitectura propuesta o es incoherente. 	0

Criterio 1h) Ayudas y subvenciones

Nivel 	Descripción 	Puntuación

Recomendaciones

    Sé realista: Es mejor un proyecto pequeño bien terminado que uno ambicioso incompleto.
    Valida tu idea: Antes de documentar todo, comparte un resumen con el profesorado.
    Piensa en el MVP: Define el mínimo necesario para que el proyecto sea funcional y útil.
    Investiga el mercado: Conocer la competencia te ayudará a diferenciarte.
    Planifica con margen: Incluye tiempo para imprevistos en tu cronograma.
